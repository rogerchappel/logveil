# Roadmap

## 0.1 MVP

- Local file and directory scanning.
- Markdown repro bundles.
- JSON audit reports.
- Default redaction rules for common secrets and identifiers.
- CI-friendly `--fail-on` gates.

## 0.2 Rule configuration

- User-supplied rule files.
- Per-project allowlists for safe paths and domains.
- Snapshot tests for fixture output.

## 0.3 Integrations

- SARIF export.
- GitHub Actions examples.
- Agent workflow recipes for sharing sanitized failures.
